Задача - создать веб-приложение, в котором будут реализованы шифратор и дешифратор.
  1. Должна быть возможность менять тип шифрования
  2. Должна быть возможность сменять темы (светлая, темная)

  1 этап
    1. Создать Шифраторы и дешифраторы
      а. в js файлах написать код

  2 этап
    1. Создать html разметку
      а. Создать Шапку
        i. Заголовок
        ii. Навигация по переключению между типами шифрования
        iii. Иконка по смене темы
      б. Создать Основную область с контентом
        i. Создать контейнер в котором будут шифратор и дешифратор
        ii. Создать два контейнера (первый - шифратор, второй - дешифратор)
        iii. Разместить по местам контент в этих двух контейнерах
      в. Создать Подвал
       i. Создать копирайт (Имя, фамилию, группу, год)

  3 этап
    1. Добавить css стили
      а. Общие стили
      б. Стили для страницы с шифром Цезаря
      в. Стили для страницы с шифром Вернама
